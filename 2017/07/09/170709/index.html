发表于 2017-07-09 | 分类于 生活

\

终于搭建起了属于自己的小天地\

突发奇想

最近刚放暑假,脑子里一堆想要做的事情,现在第一件想要做的事情——搭建一个自己的博客,总算是大体完成了.感觉还有很多东西要做,但是还是决定以后慢慢来一步一步完善这个心血来潮做出来的小东西

过程

建站的过程还是有点曲折的,虽然不是很麻烦,网上也有很多的教程,但是实际操作中还是有很多的坑啊,对于我这个只是稍微学过C语言的人还是有点懵,所以后面基本是出什么错误,直接粘贴复制百度,一点一点总算是完成了

想要做的事

搭建这个博客,也是想给自己一个专门用来写东西的地方吧,虽然自己写东西没啥文采,但还是喜欢自己瞎写一点,数量也不算多,也曾经试过各种各样的软件,但总觉得自己写的东西不在自己手上很没有安全感,所以想弄一个博客的想法,也时常会蹦出来,所以,趁着刚放假,就建了一个
这个暑假要做一个STITP的项目,所以最近在狂学ZigBee,我尽量把我学的一些东西也发上来

最后说明

如果有某位高人不幸误打误撞进了我的小站,对于我所写的东西颇有不满的话,还请多多见谅,毕竟也算缘分嘛
